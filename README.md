IIoT-Based Smart HVAC Control System for Industrial Halls

Project Overview:
This project aims to design and develop a smart HVAC system using IIoT technologies to monitor and optimize environmental conditions in industrial halls.

 Objectives:
- Monitor temperature, humidity, air quality, and airflow
- Analyze thermal comfort and heat load
- Implement smart airflow control
- Improve energy efficiency

System Architecture:
Sensors → ESP32 → Control Logic → Fan → Cloud → Dashboard

Prototype Description:
A scaled industrial hall model is used with distributed sensors to measure environmental parameters and control airflow using intelligent logic.

Proof of Concept:
The system demonstrates:
- Temperature variation across zones
- Heat load simulation
- Airflow impact on cooling
- Automated control response

Repository Structure:
- `/code` → Microcontroller programs  
- `/data` → Sample sensor data  
- `/dashboard` → IoT configuration  
- `/docs` → System design  
- `/tests` → Testing scripts  

Tools & Technologies:
- ESP32  
- DHT22 Sensor  
- MQ135 Sensor  
- Anemometer  
- Arduino IDE  
- ThingSpeak
